# ATM Management System (Python)

##  Project Description
The ATM Management System is a menu-driven Python application that simulates basic ATM operations.
It allows users to check their balance, deposit money, and withdraw money securely using simple logical conditions.

This project is ideal for beginners to understand **functions, conditionals, loops, and global variables** in Python.

---

##  Features
- Check account balance
- Deposit money into account
- Withdraw money with balance validation
- Prevents invalid transactions
- Simple and interactive menu-driven interface
- Runs continuously until user exits
